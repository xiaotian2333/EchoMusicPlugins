# 封面兜底

自定义 EchoMusic 中无封面或封面加载失败时的显示内容。

## 功能

- 文字渲染：生成轻量 SVG 封面，可设置主文字、副文字和字号，颜色自动跟随 EchoMusic 当前主题色。
- 自定义图片：支持本地图片文件或图片地址。
- 支持主窗口和 mini 播放器。

## 设置

在 EchoMusic 插件管理中打开插件卡片的设置。本插件使用自定义 Vue 组件渲染设置面板，支持预览、兜底类型切换、本地图片选择和文字参数调整。

本地图片会通过 EchoMusic 插件文件 API 转换为安全的 `file://` 地址；文字模式使用 `data:image/svg+xml`，不会额外请求网络。

文字模式的背景色来自 EchoMusic 当前主题色，文字颜色会自动按明暗计算，不需要单独配置背景色、强调色或文字色。
