# 自定义图标 (custom-icon)

自定义托盘图标、任务栏图标、桌面快捷方式图标、启动画面以及启动音效。

## 功能

### 图标

- **托盘图标**：自定义系统托盘区域显示的图标（支持 `.png`、`.ico`）
- **任务栏图标**：自定义任务栏及窗口左上角显示的图标（推荐 `.ico` 格式，需固定到任务栏）
- **桌面快捷方式图标**：自定义桌面快捷方式指向的图标（必须使用 `.ico` 格式）

### 启动画面

- 自定义启动画面背景图片（支持 `.png`、`.jpg`、`.webp`、`.gif`）
- 显示时长可调（0.5\~30 秒，默认 3 秒）
- 图片适配方式：裁剪铺满 (cover)、完整显示 (contain)、拉伸填充 (fill)
- 暗化叠加层：可调节透明度和颜色
- 图片模糊效果（0\~20px）
- 背景兜底颜色
- 显示/隐藏 LOGO 开关

### 启动音效

- 自定义应用启动时播放的音效（支持 `.mp3`、`.wav`、`.ogg`、`.m4a`、`.flac`、`.aac`、`.wma`）
- 播放音量可调（0\~100%）
- 播放时长可调（0.5\~30 秒，默认 3 秒）
- 支持试听功能

## 系统要求

- EchoMusic >= 2.2.7-beta.1

## 使用说明

1. 在插件设置中开启「开启自定义图标」
2. **图标页**：分别选择托盘、任务栏、桌面快捷方式图标，点击「保存」
3. **启动画面页**：选择启动画面图片，调整显示时长和外观设置，点击「保存」
4. **启动音效页**：选择启动音效文件，调整播放音量和时长，点击「保存」
5. 点击「恢复默认」可还原对应页面的设置并删除已复制的文件

## 备注

- 任务栏图标必须固定到任务栏，否则图标会在取消固定后失效
- 桌面快捷方式图标仅支持 `.ico` 格式
- 由于软件限制，GIF 启动画面只能显示约 1 秒，无法完整播放动画
- 插件启用/禁用时不会删除已复制的图标文件，仅在更换或恢复默认时清理
- 启动音效播放时长请根据实际音效长度设置，避免播放不完整
- 启动画面中的部分功能来自群友@小栀（rinnki）
