# Apple Music 歌词弹跳

为 EchoMusic 页面歌词添加接近 Apple Music 的行切换弹簧回弹动画。

## 功能

- 使用 `ctx.lyricEffects.register()` 接入宿主页面歌词动效扩展点。
- 不加额外氛围层，不接管歌词渲染。
- 接管页面歌词自动跟随滚动，用弹簧驱动 `scrollTop`，让整列歌词追随目标行时产生回弹。
- 当前行附近仍会做轻微缩放/淡入淡出，强化焦点切换，但不替换宿主逐字歌词。
- 切换目标时保留当前速度并重建弹簧曲线，避免普通缓动的机械感。
- 根据相邻歌词的时间间隔动态调整弹簧强度，快节奏歌词会更灵敏。
- 当前行保持原尺寸，非当前行可轻微缩小，以突出焦点。
- 设置面板可调整弹跳力度、当前行缩放、阻尼、影响范围，并可开关弹簧滚动。
- 遵守系统“减少动态效果”偏好，会降低回弹幅度。

## 兼容性

需要 EchoMusic `>=2.2.7-beta.30`，因为插件依赖页面歌词自动滚动接管扩展点。
